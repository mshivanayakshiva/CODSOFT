# CodSoft Data Science Internship

This repository contains the tasks completed during my Data Science Internship at CodSoft.

## Tasks

### Task 1: Movie Genre Classification
Build a machine learning model to classify movies into different genres based on their plot summaries using Natural Language Processing (NLP) techniques.

### Task 2: Credit Card Fraud Detection
Develop a machine learning model to detect fraudulent credit card transactions and distinguish them from legitimate transactions.

### Task 3: SMS Spam Detection
Create a machine learning model that classifies SMS messages as Spam or Ham (Not Spam) using text classification techniques.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- NLP (Natural Language Processing)
